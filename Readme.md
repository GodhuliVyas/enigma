Deploying server

model-server contains the server

Create Virtual Environment in the model-server if you want and activate it with

```bash
python -m venv .venv
```

1. ```bash
   pip install -r requirements.txt 
```

2. Run the Uvicorn server
```bash
uvicorn main:app
```
